[Instrucciones de Ejecución]

-----------------------------------------------------------------------------------------------------------
Para lograr una exitosa ejecución del programa, en la terminal, se deben tener en cuenta los 
siguientes comandos:
1-"make cityDefender": Este comando compila el programa.

2-"make run" o "make runvalgrind": Este comando ejecuta el programa para que se pueda utilizar, la primera
opción es una ejecución normal y la segunda es para comprobar que no haya leaks de memoria.

3-"make remove": Este comando elimina el programa que se compiló de su computadora para que no ocupe espacio.
